Featured project

Movie Match

Winners of GDSC Bit'N'Build Kerala State Hackathon

Movie-Match boasts an extensive movie database, housing a vast collection of films from various genres, languages, and eras. The database is updated daily using a python script and gets the daily dataset released by IMDB, uses api calls to OMDB and TMDB and updates the MongoDB database everyday. Thus, the database has the potential to host every single movie present on IMDB.

(As of 06-02-2024 - Database has data from 1940 to mid 2023 in no particular order)

Movie-Match, powered by GPT-4-turbo, revolutionizes movie recommendations with personalized suggestions. Seamlessly explore films via intuitive search, filtering by genre, language, or release year. Detailed movie info pages feature reviews, IMDB links, and streaming service details.

Tech Stack: HTML, CSS, Javascript, Python, NodeJS, FuseJS, OpenAI GPT-4-turbo

Featured project

Harvest Buddy

Finalists at GDSC Bit'N'Build International Hackathon, CRCE Mumbai

Harvest Buddy, epitomizes innovation in agriculture. Developed in 24 hours, this software solution revolutionizes farming by integrating smart inventory management and real-time alerts. Through SMS notifications, farmers receive updates on inventory consumption and crop growth, while AI-driven recommendations optimize produce utilization and assess market value.

Harvest Buddy offers a comprehensive solution for agricultural needs. With real-time weather and disaster updates, along with a coordinate system from OpenMapBox, it ensures farmers are equipped with the tools and insights needed to navigate the dynamic landscape of agriculture effectively.

Tech Stack: Javascript, NodeJS, OpenMapBox, OpenAI GPT4, Twilio

FLiX

Flix is a movie recommender which uses the MovieLens database to provide movie recommendations based on user preferences. The app employs a collaborative filtering technique using SciKit-Learn and Python. Built with ReactJS on Vite with MySQL, nodeJS and fastAPI servers, flix is super fast and easy to use.

Tech Stack: Python, ReactJS, Vite, NodeJS, fastAPI, Numpy, Pandas, SciKit-Learn, MathPlotLib, MySQL
